∀p∈P,Zp​×Zp​={(a,b):a,b∈Zp​}

Does (Zp​×Zp,+,.) define a field? If yes then how to define the addition and multiplication operations?

Let’s look into Zp​.
(Zp​,+)=(Z/pZ,+) defines a cyclic group under modular addition and also (Zp∗​,.) under modular multiplication where Zp∗​={a∈Zp​:gcd(a,p)=1} and ∣Zp∗​∣=φ(p) (in this case it’s the integers from 1 to p-1).
And also we know (Zp​,+,.) defines a field unlike (Z,+,.) which defines an integral domain, it’s easy to prove.

Theorem: Let (Zp​,+,.) be a field, then there are atleast ⌊2p​⌋ irreducible polynomials like p(x) in Zp​[x] s.t. deg(p(x))=2.
Proof.

  • p=2:
    there exists only x2+x+1.

    We said the polynomials should have no root in Zp​, so let’s remove the integers like a∈Zp∗​ s.t. a∗a=ω2.
